# 機械学習でよく使うコード集
## グリッドサーチ

```python
from sklearn.model_selection import ParameterGrid

all_params = {
    'params1':[10, 20, 30, 40],
    'params2':[0.5, 0.7, 1.0],
    'params3':[2, 4, 6],
    'params4':['accuracy', 'f1']
    
}

for params in tqdm(list (ParameterGrid(all_params))):
    Classfier(**params)
```
